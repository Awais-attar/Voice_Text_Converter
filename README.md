# 🎙️ Text-to-Voice and Voice-to-Text Converter

A Python desktop application that converts **text into speech** and **speech into text** through a simple and user-friendly graphical interface. The application is built using **Tkinter** and supports voice customization, file management, and clipboard operations.

## ✨ Features

* 🔊 Convert text to voice
* 🎤 Convert voice to text
* 📂 Open text files
* 💾 Save text files
* 📋 Copy text to clipboard
* 🗑 Clear text
* 👤 Select different voices
* 🎚 Adjust speech speed
* 🔊 Adjust volume
* 🧵 Responsive interface using multithreading

## 🛠️ Technologies Used

* Python
* Tkinter
* SpeechRecognition
* pyttsx3
* sounddevice
* SciPy

## 🚀 Installation

1. Clone the repository:

```bash
git clone https://github.com/Awais-attar/Voice_Text_Converter.git
```

2. Move to the project folder:

```bash
cd Voice_Text_Converter
```

3. Install the required packages:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
python gui.py
```

## 📖 Usage

* Enter text and click **Speak** to convert text into speech.
* Click **Listen** and speak through your microphone to convert speech into text.
* Use **Open** and **Save** to manage text files.
* Use **Copy** and **Clear** for quick text operations.
* Customize the voice, speech rate, and volume using the available controls.
